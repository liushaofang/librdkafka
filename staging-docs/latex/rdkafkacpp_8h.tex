Apache Kafka C/C++ consumer and producer client library. \hyperlink{rdkafkacpp_8h}{rdkafkacpp.h} contains the public C++ API for librdkafka. The API is documented in this file as comments prefixing the class, function, type, enum, define, etc. For more information, see the C interface in \hyperlink{rdkafka_8h}{rdkafka.h} and read the manual in INTRODUCTION.md. librdkafka version Interpreted as hex {\ttfamily MM.mm.rr.xx}:
\begin{DoxyItemize}
\item MM = Major
\item mm = minor
\item rr = revision
\item xx = pre-\/release id (0xff is the final release)
\end{DoxyItemize}

E.g.: {\ttfamily 0x000801ff} = 0.8.1

\begin{DoxyRemark}{Remarks}
This value should only be used during compile time, for runtime checks of version use RdKafka::version() 
\end{DoxyRemark}
